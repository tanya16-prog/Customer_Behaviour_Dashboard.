# Customer_Behaviour_Dashboard. 
- Project Title / Headline

  Customer Behaviour Analysis Dashboard | Python, MySql,  Power BI Business Intelligence Project,
 - Short Description / Purpose

  The Customer Behaviour Analysis Dashboard is an interactive Power BI project designed to analyze customer purchasing patterns, sales performance, and demographic trends. The dashboard provides valuable insights into customer behavior by tracking sales, revenue, customer segments, age groups, gender distribution, and purchase trends.

It helps businesses understand customer preferences, identify high-value customers, and support data-driven decision-making through interactive visualizations.

- Data Source

  The dashboard is built using a Customer Behaviour/Sales dataset containing customer and transaction information such as:

Customer ID
Customer Demographics (Age, Gender)
Product Information
Purchase Date
Sales Revenue
Customer Segment
Purchase Frequency
Order Details
Sales Transactions
